# TypeScript Fundamentals 🚀

A comprehensive TypeScript learning repository covering core concepts from beginner to intermediate level through hands-on examples and notes.

## 📚 Topics Covered

### Fundamentals

- Type Inference
- Primitive Types
- Special Types (`any`, `unknown`, `never`, `void`)
- Objects
- Literal Types

### Type System

- Type Assertions
- `as const`
- Union Types
- Intersection Types

### Collections

- Arrays
- Readonly Arrays
- Tuples

### Functions

- Parameter Annotations
- Optional Parameters
- Default Parameters
- Rest Parameters
- Tuple Rest Parameters
- Return Types

### Object Modeling

- Interfaces
- Type Aliases
- Interfaces vs Types
- Index Signatures
- Record Utility Type

### Type Narrowing

- `typeof`
- `instanceof`
- `in` Operator
- Optional Chaining (`?.`)
- Nullish Coalescing (`??`)

### Generics

- Generic Functions
- Type Parameters
- Type Inference
- Generic Type Aliases
- Generic Interfaces

## ⚙️ Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate into the project:

```bash
cd TYPESCRIPT
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Compile TypeScript

Generate JavaScript files:

```bash
npx tsc
```

or

```bash
npm run build
```

Generated files will be placed inside:

```text
dist/
```

---

## ▶️ Run Compiled Files

Example:

```bash
node dist/01_Inference.js
```

Run any compiled lesson:

```bash
node dist/<filename>.js
```

## 🧠 Learning Approach

This repository follows a concept-by-concept learning strategy:

1. Learn the concept.
2. Write practical examples.
3. Understand TypeScript inference.
4. Explore edge cases.
5. Add notes and explanations.
6. Build a strong foundation before moving to advanced topics.
